Tentang Aplikasi Data Cuaca

Aplikasi ini adalah REST Client sederhana yang memanfaatkan OpenWeatherMap API untuk menampilkan informasi cuaca berdasarkan nama kota.

Tujuan utama aplikasi:

  • Melatih penggunaan HTTP request (fetch) ke Open Public API.
  • Memahami struktur data JSON dan cara menampilkannya ke antarmuka web.
  • Menerapkan konsep REST Client dalam bentuk aplikasi web sederhana.

Fitur yang tersedia:

  1. Beranda – Pencari cuaca instan berdasarkan nama kota.
  2. Kota Populer – Menampilkan cuaca untuk beberapa kota yang sering digunakan.
  3. Riwayat Pencarian – Menampilkan daftar kota yang pernah dicari beserta ringkasan cuacanya.
  4. Tentang – Halaman penjelasan singkat mengenai aplikasi dan teknologi yang digunakan.

Teknologi yang digunakan: HTML, CSS (Bootstrap 5), dan JavaScript murni (fetch API) tanpa framework tambahan.
